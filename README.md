# AI-Powered-Company-Brochure-Generator

Description:
This project automatically generates a structured company brochure from a website using Large Language Models (LLMs).
It scrapes relevant company pages (such as About, Products, and Careers) and synthesizes the information into a concise Markdown brochure.

Key Features:

Website scraping (landing + relevant pages)

LLM-based link relevance selection

Automated brochure generation

Markdown structured output

Single-file Python implementation

Tech Stack:

Python

OpenAI API (LLM)

BeautifulSoup (web scraping)

Requests

Markdown generation

Use Case:
Automated company profiling, sales intelligence, and content generation.
